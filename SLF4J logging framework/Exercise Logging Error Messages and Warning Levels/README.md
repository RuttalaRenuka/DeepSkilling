# Logging Example using SLF4J

## Description
This project demonstrates logging in Java using the SLF4J API with the Logback implementation.

## Technologies Used
- Java 23
- Maven
- SLF4J 1.7.30
- Logback 1.2.3

## Features
- Logs ERROR messages
- Logs WARN messages

## Output

```
ERROR LoggingExample - This is an error message
WARN LoggingExample - This is a warning message
``
