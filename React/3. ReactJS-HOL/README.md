
# Student Score Calculator App

## Objective
This React application demonstrates the use of Function Components and Props to calculate and display the average score of a student.

## Technologies Used
- React
- JavaScript
- CSS
- Node.js
- npm

## Folder Structure

```
src
├── Components
│   └── CalculateScore.js
├── Stylesheets
│   └── mystyle.css
├── App.js
└── index.js
```

## Features

- Function Component
- Props
- CSS Styling
- Average Score Calculation

## Run the Project

```bash
npm install
npm start
```

Visit:

```
http://localhost:3000
```
